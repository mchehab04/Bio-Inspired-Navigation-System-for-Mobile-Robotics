# Python Utility Scripts

## Overview
This folder contains Python scripts used for debugging and validating hardware communication outside of Promethe2.

---

## Scripts

| File | Description |
|---|---|
| `rc_to_js.py` | Converts RC joystick packets to joystick events |
| `read_distance.py` | Reads sensor distance values |
| `read_joystick_packets.py` | Debugs joystick serial packets |
| `read_lidar.py` | Reads and validates LIDAR data |

---

## Purpose

These scripts were used for:
- Hardware debugging
- Packet inspection
- Serial communication testing
- Sensor validation
