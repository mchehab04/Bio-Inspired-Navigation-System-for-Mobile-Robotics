# Joystick Demo

## Overview
This folder contains scripts used to test joystick communication and manual robot control.

---

## Features Tested

- RC100/BT-410 packet decoding
- Serial communication
- Manual robot navigation
- Promethe2 joystick integration

---

## Important Files

| File | Description |
|---|---|
| `joystick_new.script` | Main joystick test script |
| `joystick.hwj` | Joystick configuration |
| `arduino_port.hws` | Serial communication settings |

---

## Notes

This subsystem was used to manually control the robot and validate directional commands before autonomous navigation integration.
