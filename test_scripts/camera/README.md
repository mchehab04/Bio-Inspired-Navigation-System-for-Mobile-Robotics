# Camera Test

## Overview
This folder contains scripts used to validate camera acquisition and image processing in Promethe2.

---

## Features Tested

- USB camera capture
- FireWire camera configuration
- Image acquisition
- Promethe2 visual processing compatibility

---

## Important Files

| File | Description |
|---|---|
| `camera_usb.script` | Main camera test script |
| `camera_usb.config` | Configuration file |
| `camHD.hwc` | HD camera configuration |
| `capture.hwc` | Camera capture configuration |
| `firewire.hwc` | FireWire camera configuration |

---

## Notes

These scripts were used before integrating camera vision into the navigation system.
